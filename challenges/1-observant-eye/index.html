üîç What is Image Metadata?

Metadata is "data about data" - hidden information stored within files that describes the file itself. This includes:

  • Camera settings (for photos)
  • Date and time created
  • GPS coordinates (for photos)
  • Author information
  • Custom comments and descriptions

In this challenge, the flag is hidden in one of these metadata fields!
