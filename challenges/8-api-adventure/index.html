"We built a simple book API. Your goal is to find the details of the book with ID 0. The endpoint is provided below."

üéØ Your Mission:

APIs (Application Programming Interfaces) allow programs to communicate with each other. Your task is to query the book API endpoint to find information about a specific book.

üìö Book API Endpoint:

GET https://crypton-ctf-platform.onrender.com/api/books/0

This API returns book information in JSON format. Try accessing it to find the flag!

üí° Hint:

Websites often provide data through API endpoints. You can request information directly from these endpoints using various tools.

üîß How to Access the API:

Web Browser (Easiest):

Simply visit the endpoint URL in your browser:

https://crypton-ctf-platform.onrender.com/api/books/0

Command Line (curl):

curl https://crypton-ctf-platform.onrender.com/api/books/0

JavaScript (Browser Console):

fetch('https://crypton-ctf-platform.onrender.com/api/books/0')
.then(response => response.json())
.then(data => console.log(data));

Python:

import requests
response = requests.get('https://crypton-ctf-platform.onrender.com/api/books/0')
print(response.json())

üîç Expected Response Format:

The API will return JSON data that might look like:

{
  "id": 0,
  "title": "Book Title",
  "author": "Author Name",
  "flag": "CRYPTON{...}"
}

üìñ About REST APIs:

REST (Representational State Transfer) APIs use standard HTTP methods to perform operations:

  • GET: Retrieve data (used in this challenge)
  • POST: Create new data
  • PUT: Update existing data
  • DELETE: Remove data

APIs are the backbone of modern web applications and mobile apps!
