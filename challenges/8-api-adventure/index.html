"We built a simple book API on our CTF platform. Your goal is to explore the API and find information about the book with ID 0."

üéØ Your Mission:

APIs (Application Programming Interfaces) allow programs to communicate with each other. Your task is to discover and query the book API endpoint on our CTF platform to find information about a specific book.

üîç API Discovery:

The book API is hosted on our main CTF platform. You'll need to:

Step 1: Find the API endpoint URL
Step 2: Make a request to get book information
Step 3: Extract the flag from the response

Hint: APIs often follow predictable URL patterns...

üí° API Hint:

Websites often provide data through API endpoints that follow REST conventions. Common patterns include:

  • /api/books - Get all books
  • /api/books/1 - Get book with ID 1
  • /api/books/0 - Get book with ID 0

Think about what endpoint would return information about book ID 0.

üîß Tools to Access APIs:

Web Browser (Easiest):

Simply visit the API endpoint URL in your browser. APIs return data that browsers can display.

Command Line (curl):

curl https://your-target-url/api/endpoint

Browser Developer Tools:

Use the JavaScript console to make API requests:

fetch('/api/books/0')
  .then(response => response.json())
  .then(data => console.log(data));

Python Script:

import requests
response = requests.get('/api/books/0')
print(response.json())

Tools like Postman or Insomnia:

Professional API testing tools that provide nice interfaces.

üìñ Expected Response Format:

APIs typically return data in JSON format. The response might look like:

{
  "id": 0,
  "title": "Book Title",
  "author": "Author Name",
  "flag": "CRYPTON{...}"
}

Look for the flag field in the JSON response!

üîé How to Find API Endpoints:

  • Check the website's main domain (our CTF platform)
  • Look for common API paths like /api/, /v1/, /rest/
  • Browser Developer Tools ‚Üí Network tab can show API calls
  • Sometimes mentioned in HTML comments or JavaScript files
